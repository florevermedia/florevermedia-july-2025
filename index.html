About Florevermedia

Florevermedia is a web design and development studio led by Manuela Flores — a Bolivian creative with a background in marketing agencies and social media strategy. Every website we build is tailored to the client’s needs, combining thoughtful design, technical precision, and SEO best practices.

What sets Florevermedia apart is our commitment to accessibility and ease of use. We design and personalize each website so that our clients can update it confidently — without worrying about breaking anything. And because we work in both English and Spanish, our websites help businesses connect with their audience — without language barriers.

Around the Web

Made with ❤️ from La Paz, Bolivia

Repair the Multiverse

Repair the Multiverse is a game where you are a hero who is repairing the whole multiverse, with different creatures. It was developed in Unity. For the illustrations we used Photoshop Illustrator and for the soundtrack was FL Studio 20. Manuela brought in game developers and designers, working very closely with them to create and publish this game. She helped with the coding, graphic design, the game concept design, and the music composition.

Kaizen

Kaizen is a game about a girl that has a lot of fears. The three bosses represent her 3 biggest fears. At the end of the game, the main character finds peace with herself. Manuela brought in game developers and designers, working very closely with them to create and publish this game. She helped with the coding, the game concept design, and the music composition. It was our entry for Global Game Jam 2020.

The Video Games as Mental Health Therapy Project

Video Games as Therapy for the Bolivian Population was the university presentation that launched Manuela's most ambitious goal and dream — to combine video games and therapy for a revolutionary advance in mental health treatment. The presentation linked is current in Spanish. Contact Manuela if you would like to see the English version.

MakeUp Memory

MakeUp Memory is a basic 5-level game in which the player will have to challenge their memory by clicking on the cards and seeing the different emojis that it has until they match with the same one. There are two moods in this game, they are "relax" and "normal". In the first one like the name suggests the player can select whichever level that their want and take as moves and time as they want, which is like a practice mood, in the second one we count the movements and the time. The game ends if the player is with 0 of any of this he/she will have to start from level 1. Here you can see the code in HTML, CSS and JavaScript if you would like to see how Manuela made this.
